jQuery Wiggle

About

The Wiggle plugin is a script that offer some of the functionality of the Effect library from jQuery UI. It is an alternative, useful in those situations when you need only one effect and do not wish to include the entire jQuery UI library in your project.

For example, the Blink plugin weighs in at 576 bytes. Comparatively, the jQuery UI core plus Effects library weigh in at around 125 kb.

Demo



Basic Usage

Include the jQuery library and the wiggle plugin:

<script type="text/javascript" src="javascripts/jquery-1.3.2.min.js"></script>
<script type="text/javascript" src="javascripts/jquery.wiggle.min.js"></script>

Then, just call wiggle() on the desired element.

$(".selector").wiggle();

Additional Parameters

$(".selector").wiggle({
	speed: [speed of animation (ms); default: 50],
	wiggles: [number of times to wiggle; default: 3],
	travel: [number of pixels to move],
	callback: [function]
});

Note: Callbacks are performed once per matched element.
